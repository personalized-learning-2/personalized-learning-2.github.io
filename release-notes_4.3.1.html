PLUS 4.3.1 Release Notes

9/2/2022

In time for the start of a new school year, we are releasing the latest updates to the PLUS app. The functionality has not changed but it has a fresh new look!

4.3.1 Update List

  Branding Updates

  • With this latest release we have updated the app to reflect our new branding! Look for these changes on the static site soon.

  Bug Fixes
  • Independent users must now specify the name of their organization when creating an account.
