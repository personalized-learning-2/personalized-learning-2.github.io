4.3.0 Update List

  PLUS Training - Resources Features Updates

  • Lessons are now available to users under the age of 18. Data from these users will not be included in research efforts.

  • Resource links in lessons are now displayed as cards, which include the resource description. Also, from here the resources can be pinned for ease of future access.

  Minor Updates

  • Response options for multiple-choice questions are now randomized on Lessons that haven't been completed. This encourages active processing of material and reduces the ability of learners to "game" the lesson.

  • Improved the resolution and readability of tables in lessons.

  • Breadcrumbs are enabled on individual lessons so users can know which lesson they're in once they've left the title page.

  Bug Fixes
  • Fixed a problem with the Next button that allowed users to submit incomplete lessons.
  • Fixed a problem where lessons with certain survey question responses flagged the submitted lesson as incomplete.
